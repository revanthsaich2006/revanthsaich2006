### 🚀 Featured Projects

📂 **[StudySphere - AI-Powered Learning Progress Tracker](https://github.com/revanthsaich2006/STUDYSPHERE)**  
*A comprehensive web application designed to help students track learning progress, log study sessions, set academic goals, and receive AI-generated recommendations.*
*   🧠 **AI-Powered Suggestions**: Integrates Azure OpenAI to provide personalized study suggestions and actionable recommendations based on history.
*   🎙️ **Voice Reflections**: Features audio recording and playback capabilities allowing students to capture verbal reflections on study habits.
*   📈 **Analytics & Live Dashboard**: Implements interactive progress charts and study streak trackers with real-time updates powered by Flask-SocketIO.
*   🔒 **Secure Architecture**: Built with JWT authentication, role-based access, and structured backend REST APIs linked to SQLite/PostgreSQL.

📂 **[Restaurant Management System GUI](https://github.com/revanthsaich2006/Restraunt-Management-System-GUI)**  
*A complete desktop application built in Java for managing restaurant operations.*
*   🔑 **Multi-role Auth**: Separate dashboards for Admins (analytics, menus, tables) and Waiters (ordering, seating).
*   🪑 **Interactive Seating**: Dynamic table booking status, tracking active vs. completed sessions.
*   📊 **Analytics & Low Stock Alerts**: Background threads monitoring stock levels with automatic logging.
*   🧾 **Auto-Generated PDFs**: Automatically compiles orders and compiles bills into professional PDFs using `OpenPDF`.
